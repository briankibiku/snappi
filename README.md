# Snappi Website Development Prompt

## Project Overview
Build a modern, professional SaaS website for **Snappi IT Systems** - a customer-first CRM system for restaurants and hospitality businesses. The website should showcase the product, explain the solution, and drive conversions.

## Design Inspiration
Use **Odoo.com** as the primary design inspiration:
- Clean, modern, and professional layout
- Clear value propositions above the fold
- Feature sections with icons and descriptions
- Smooth animations and transitions
- Strong call-to-action buttons
- Trust indicators (testimonials, stats, logos)
- Mobile-first responsive design

## Product Description

Snappi IT Systems is a customer-first CRM system that promotes service efficiency by:

- Promoting increased sales

### Target Audience
Restaurants, bars, hotels, and hospitality businesses looking to:
- Reduce service delays and lost orders during peak hours
- Improve cash collection and reduce leakage
- Implement professional staff management
- Gain better business visibility and information flow

## Design Requirements

### Visual Style
- Modern, clean, and professional
- Use subtle gradients with the orange color palette
- Rounded corners for cards and buttons (8-12px)
- Generous white space
- Shadow effects for depth (subtle)
- Smooth transitions and hover effects

### Components to Include
- Navigation bar (sticky)
- Hero section with CTA
- Feature cards with icons
- Stats/metrics display
- Pricing table
- Testimonial cards
- FAQ accordion
- Contact form
- Footer

### Animations
- Fade-in on scroll
- Hover effects on buttons and cards
- Smooth transitions between sections
- Loading states for forms

### Responsive Design
- Mobile-first approach
- Breakpoints: 640px (sm), 768px (md), 1024px (lg), 1280px (xl)
- Stack columns on mobile
- Hamburger menu for mobile navigation

### Call-to-Action Buttons
**Primary Button Style:**
- Background: #FF9B6A (Primary Orange)
- Text: White
- Padding: 16px 32px
- Border radius: 8px
- Font weight: 600
- Hover: Slightly darker orange (#FF8A5C)
- Shadow on hover

**Secondary Button Style:**
- Border: 2px solid #FF9B6A
- Text: #FF9B6A
- Background: Transparent
- Hover: Background #FF9B6A, Text White

### Icons
- Use Heroicons, Lucide, or similar icon library
- Consistent size and style throughout
- Orange color for primary icons

## Technical Requirements

### Tech Stack
- Next.js 14+ (App Router)
- Vite
- React 18+
- TypeScript
- Tailwind CSS
- Framer Motion (for animations)

### Performance
- Optimize images (use Next.js Image component)
- Lazy load sections
- Fast page load time (<2s)
- Lighthouse score >90

### SEO
- Meta tags for all pages
- Open Graph tags
- Schema markup for business
- Semantic HTML

### Accessibility
- ARIA labels where needed
- Keyboard navigation
- Color contrast compliance (WCAG AA)
- Alt text for all images

1. Initialize Next.js project with Vite
2. Set up Tailwind CSS with custom colors
3. Create component structure
4. Set up routing

### Phase 2: Build Core Pages
1. Build homepage sections one by one
2. Create reusable components
3. Add responsive design
4. Implement animations

### Phase 3: Polish & Optimize
1. Add micro-interactions
2. Optimize performance
3. Test on multiple devices
4. SEO optimization

### Phase 4: Content & Launch
1. Add final content

3. Final review

## Notes for Developer
- Prioritize mobile experience (most users will be on mobile)
- Keep load times fast (critical for hospitality industry)
- Make CTAs prominent and easy to find
- Use real restaurant imagery (or high-quality stock photos)
- Ensure forms are simple and quick to fill
- Add subtle animations but don't overdo it
- Test thoroughly on different devices
